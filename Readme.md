####系統簡介####
隨著科技不斷的進步，越來越多人開發APP的應用程式，但尋找朋友定位的APP有些應用程式不完全，搜尋結果是零的情形，沒有什麼幫助。
因此我們設計一套使用者以FB的帳號登入並同意取得權限，可以透過FB的好友查詢好友定位與Google Map做連結的APP應用程式。
